# Teste SMARKIO
Repositório para o teste da vaga de programadora de chatbot e desenvolvedora de assistentes virtuais.

Sobre os arquivos:  config.js se encontra dentro de uma pasta específica para o banco de dados, chamada database
                    comentariosController.js está na pasta src
                    app.html e style.css estão dentro da pasta front que é uma subpasta de src
                    routes.js e server.js estão na pasta raiz do teste
                    
Utilizei vs.code para a criação e edição e o sistema operacional Linux.
